# Quick-Hexo

> 其实就是一个为了使用方便而写的辣鸡Python脚本..

## 安装(获取)

```
wget https://raw.githubusercontent.com/wzk0/quick-hexo/main/hexo.py
```
> 以及requests依赖

## 使用

```
python hexo.py
```

> 注意，要在博客根目录下运行.
